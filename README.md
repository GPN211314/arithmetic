# arithmetic
其中主体部分用python实现，而扩展部分选择用haskell语言实现

```tcl
/ haskell
	/ Calculate.hs
	/ Create.hs
	/ Main.hs
	/ Main.exe
/ python
	/ Generate.py
	/ Calculate.py
	/ Main.py
	/ cal_test.py
	/ gen_test.py
	/ arithmetic.exe
```

##### Main.exe/arithmetic.exe       [选项]

##### [选项]：

##### 	-c  用户选择乘方符号，生成1000道不重复的四则运算题目

##### 	-s  用户输入结果，判断对错，输入为空时退出并给出统计结果	

##### 	-h  显示当前信息
